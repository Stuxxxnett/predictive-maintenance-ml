Predictive Maintenance Dashboard

Machine learning powered failure risk prediction

This tool predicts whether an industrial machine is at risk of failure based on its current sensor readings. It uses a Random Forest model trained on the AI4I 2020 Predictive Maintenance Dataset, which contains 10,000 real operating records labeled with actual failure outcomes.

How it works: The model looks at five sensor measurements plus the product quality type, and learns the patterns that typically precede a failure (most importantly torque, rotational speed, and tool wear). Enter your machine's current readings below and the model will estimate the probability of failure under those conditions.

Note: This is a demonstration tool trained on a synthetic dataset and should not be used for real safety-critical decisions.

Enter Sensor Readings

Ambient air temp around the machine, in Kelvin. Typical range: 295-304 K
Internal process temp, in Kelvin. Typically ~10 K above air temp
Spindle/motor speed in RPM. Typical range: 1200-2900
Load on the machine, in Newton-meters. Typical range: 3-77
Cumulative tool usage time, in minutes. Typical range: 0-253
L = Low, M = Medium, H = High quality variant
